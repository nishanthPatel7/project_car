# Project Car Platform

A premium Flutter-based platform designed for high-performance ecosystem management, featuring an encrypted-first architecture and a modern, glassmorphic UI.

## 🚀 Features

- **Clean Architecture**: Organized into `core`, `features`, and `shared` for scalability.
- **Glassmorphic UI**: High-end visual aesthetics with real-time backdrop blurring.
- **Dynamic Routing**: Powered by `go_router` for seamless multi-platform navigation.
- **State Management ready**: Pre-configured with `flutter_bloc` and `provider`.
- **Premium Theming**: Custom Dark Mode system using Google Fonts (Outfit).

## 🛠️ Setup & Running

1. **Get Dependencies**:
   ```bash
   flutter pub get
   ```

2. **Run the App**:
   ```bash
   flutter run
   ```

## 📂 Project Structure

- `lib/core`: Global configurations, themes, and networking clients.
- `lib/features`: Business logic and UI for specific modules (Home, Auth, etc.).
- `lib/shared`: Reusable design tokens and widgets.
